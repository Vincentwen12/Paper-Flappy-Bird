<a id="english"></a>
## 🇬🇧 English

**Paper Flap** is a handcrafted Flappy Bird-style arcade game built entirely with procedural paper-craft aesthetics. Fold a paper bird, let it fly through a washi paper morning.

### Features

- **27 Bird Skins** — from Classic to Void, across Common / Rare / Epic / Legendary tiers
- **17 Scene Themes** — Morning Mist, Aurora Night, Sakura Blizzard, Garden of Eden...
- **25 Equipment Pieces** — Trails, Shields, and Accessories
- **5 Power-Ups** — Paper Umbrella, Paper Boat, Paper Lantern, Paper Plane, Paper Crane
- **Daily Task System** — 5–10 random tasks per day, earn Star Points
- **Japanese Zen BGM** — 4-layer procedural music (bass, chords, melody, percussion)
- **8 Languages** — 汉语 / English / 日本語 / 한국어 / Español / Français / Русский / Deutsch
- **All assets procedurally generated** — no external images needed

### Quick Start

```bash
pnpm install
pnpm dev
```

Open http://localhost:5173

### Tech Stack

- React 18 + TypeScript 5 + Vite 6
- Zustand 4 (state management)
- HTML5 Canvas 2D (game engine)
- Web Audio API (procedural synth)
- TailwindCSS 4 (styling)
- React Router DOM v6

## 📂 Project Structure

```
src/
├── components/      # UI components (GhostButton, PaperCard, PaperTitle, FitText, PreviewBird)
├── data/            # Game data (skins, themes, equipment, tasks, powerUps)
├── game/
│   ├── assets/      # Procedural drawing (bird, pipes, background, powerUps, stars)
│   ├── audio/       # Web Audio synth & AudioManager
│   └── engine/      # GameEngine, InputManager, PhysicsWorld, ParticleSystem
├── i18n/            # 8-language translation system
├── pages/           # Route pages (Home, Game, GameOver, Collection, Shop, Leaderboard, Settings)
├── store/           # Zustand stores (profileStore, gameStore)
└── utils/           # localStorage helper, random utils
```
